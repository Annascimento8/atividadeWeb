# 🌐 Atividade Web

Este repositório contém uma atividade prática de desenvolvimento web utilizando **HTML, CSS e JavaScript**.  
O objetivo do projeto é aplicar conceitos básicos de programação para web, incluindo interação com o usuário e manipulação de elementos na página.

---

# 📚 Sobre o Projeto

Este projeto foi desenvolvido como parte de uma atividade acadêmica com foco no aprendizado de:

- Estrutura de páginas web
- Estilização de elementos
- Interatividade com JavaScript
- Manipulação de entrada do usuário
- Lógica de programação aplicada ao navegador

O projeto simula uma pequena aplicação web onde o usuário pode interagir com o sistema diretamente pelo navegador.

---

# 🚀 Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas no desenvolvimento do projeto:

- **HTML5** → Estrutura da página
- **CSS3** → Estilização e layout
- **JavaScript** → Lógica e interação com o usuário
